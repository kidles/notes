## 拷贝
### 浅拷贝
- copy = [...arr/obj]
- copy = Object.assign(arr1,arr2)
- 使用其他遍历方式，都是浅拷贝
### 深拷贝
- 使用其他库
- 递归函数里遍历,遍历里判断类型
```javascript
function copy(obj){
  let res = obj instanceof Array?[]:{};
  for(let key in obj){
    res[key] = typeof obj[key]=='object'?copy(obj[key]):obj[key];
  }
  return res
}
```
